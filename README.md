# PYPA.Bankrupt


## Dependências 
 <a href="https://dotnet.microsoft.com/download">dotnet core 2.2 SDK</a>
 
## Execução Comand Line
 Entrar no diretório: 
 ``` 
 PYPA.Bankrupt\PYPA.Bankrupt 
 ```
 Executar:
 ```
 $ dotnet build
 $ dotnet run
 ```
 
## Execução Visual Studio 2019
Abrir Solução:
``` 
PYPA.Bankrupt\PYPA.Bankrupt.sln
```
 Executar projeto console: 
 ``` 
 PYPA.Bankrupt\PYPA.Bankrupt 
 ```
